## Annotations

-   ```@Component```
    -   Dependency injection 
    -   usesingle instances
-   ```@Qualifier```
    -   get Bean by name
-   ```@Rest Controller``` vs ```@Controller```
    -   REST API, JSP
-   ```RestTemplate```
    -   ```?```
-   ```@Document```
    -   Database model
-   ```@Service```
    -   Logic, DI
-   ```@Repository```
    -   DB queries
-   ```@Autowired```
    -   DI
-   ```@SpringBootApplication```
    -   Main app file
-   ```@RequestMapping```
    -   ```@GetMapping```
    -   ```@PostMapping```
    -   ```@PutMapping```
    -   ```@PutMapping```
-   ```@PathVariable```
-   ```@RequestBody```
